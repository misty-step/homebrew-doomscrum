# homebrew-doomscrum

Homebrew tap for [doomscrum](https://github.com/phrazzld/doomscrum) — PRD
brainrot triage: backlog specs as swipeable shortform video.

```bash
brew install phrazzld/doomscrum/doomscrum
```

The formula in `Formula/doomscrum.rb` is regenerated automatically by
doomscrum's release workflow on every tagged release; do not hand-edit it.
